# 💫 About Me:
Студент 3-го курса МУИТ по специальности Software Engineering. Имею практический опыт разработки веб-интерфейсов на React и Vanilla JS, а также базовые знания серверной части на Java (Spring Boot) и Firebase.<br>Ищу возможность применить свои знания в реальных проектах на позиции стажера или Junior-разработчика. Мой подход к работе: внимание к деталям, стремление писать чистый код и желание разбираться в бизнес-логике продукта. Готов быстро обучаться новым технологиям и эффективно работать в команде.<br>
